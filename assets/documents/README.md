# Resume Document

The resume PDF file is currently named `Master_resume_1.pdf` and is located in this folder.

**To update your resume:**
1. Save your updated resume as a PDF file
2. Replace `Master_resume_1.pdf` in this folder
3. Or rename it to your preferred filename and update `index.html` accordingly

The "Download Resume" button on your portfolio currently links to `Master_resume_1.pdf`.

**Note:** 
The portfolio is configured to download this file via the Netlify configuration in `netlify.toml`.
